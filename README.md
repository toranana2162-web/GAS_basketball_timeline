# BSCA CUP タイムライン自動生成ツール

Google スプレッドシート + Google Apps Script (GAS) で構築されたバスケットボール大会タイムライン自動生成ツールです。

## 概要

入力項目を埋めてボタンを押すだけで、大会タイムラインと試合スケジュールが自動生成されます。

## 機能

### 入力項目

**大会共通**

- 利用開始時刻 / 利用終了時刻
- 日数（1Day / 2Days）
- コート数（1 / 2）

**カテゴリ別（U12 / U15）**

- 開催するか（ON/OFF）
- チーム数
- 試合方式（リーグ戦 / トーナメント）
- レギュレーション（slot_minutes で管理）

### 自動生成されるもの

1. **大会タイムライン**
   - 設営
   - 開場 / 受付
   - 各試合（第1試合～第N試合）
   - エキシビジョン（任意）
   - 表彰式（任意）
   - 1〜3位の写真撮影（任意）
   - 完全撤退

2. **試合スケジュール**
   - コート割（A/B）
   - 開始時刻 / 終了時刻
   - カテゴリ区分（U12/U15）
   - チーム名（仮名でも可）
   - トーナメント表（トーナメント方式の場合）

## 自動配置ルール

- U12優先でU15と交互に試合を行う
- 連戦NG（同一チームが連続する枠に入らない、コート跨ぎでも連続NG、最低1枠空ける）
- 昼休憩は自動挿入しない
- 利用終了時刻を超えない

## レギュレーション

| ID         | カテゴリ | 1枠の時間(分) |
| ---------- | -------- | ------------- |
| U12_SLOT60 | U12      | 60            |
| U12_SLOT65 | U12      | 65            |
| U12_SLOT70 | U12      | 70            |
| U15_SLOT75 | U15      | 75            |
| U15_SLOT80 | U15      | 80            |
| U15_SLOT85 | U15      | 85            |

## 試合数計算

- **リーグ戦（5チーム以下）**: nC2（総当たり）
- **リーグ戦（6〜8チーム）**: 2グループに分かれて総当たり戦
- **リーグ戦（9チーム以上）**: 3グループに分かれて総当たり戦
- **トーナメント**: n-1

## セットアップ手順

### 1. Google スプレッドシートを作成

1. 新しいスプレッドシートを作成
2. 「拡張機能」→「Apps Script」を開く

### 2. GASコードをコピー

以下の順序でファイルを作成し、各 `.gs` ファイルの内容をコピー：

1. `Config.gs` - 設定値
2. `MatchCalculator.gs` - 試合数計算
3. `Scheduler.gs` - スケジュール最適化
4. `TimelineGenerator.gs` - タイムライン生成
5. `Main.gs` - メイン処理
6. `Setup.gs` - 初期セットアップ

### 3. 初期セットアップを実行

1. Apps Script エディタで `setupSpreadsheet` 関数を実行
2. 必要なシートと入力フォームが自動作成されます

### 4. 使用開始

1. 「入力」シートで必要項目を入力
2. 「タイムライン生成」ボタンをクリック
3. 「タイムライン」「試合スケジュール」シートに結果が出力

## ファイル構成

```
gas/
├── Config.gs           # 設定値・レギュレーション定義
├── MatchCalculator.gs  # 試合数計算ロジック
├── Scheduler.gs        # スケジュール最適化ロジック
├── TimelineGenerator.gs # タイムライン生成
├── Main.gs             # メイン処理・UI
└── Setup.gs            # スプレッドシート初期セットアップ
```

## エラー処理

- 時間内に収まらない場合は警告を表示
- 不足分（不足○分 / 不足○枠）を明示
